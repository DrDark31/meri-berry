Meri-Berry is an offline-first farm operations app for tracking fruit picker productivity and pay. Staff enter a worker number and weight at the desk, and the system records each weigh-in, auto-creates new workers if needed, and keeps running totals for weight picked, earnings, paid amount, and outstanding balance. It uses a React frontend with a Fastify + SQLite backend, designed to run locally on a single laptop without internet. It also has the feature to add specific rates for specific fruits, and export payrolls and receipts. It will also possess stats on the dashboard to show easy graphic around the specific fruits picked etc..
